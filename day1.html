Day 1 Report

Today marked the beginning of our Web Development training. We started with an introduction to the web ecosystem, including how websites work, basic HTML structure, and the role of CSS and JavaScript in modern web pages. Students were introduced to the tools required: VS Code, Chrome DevTools, and GitHub for version control.

We learned to create our first HTML page with headings, paragraphs, and links. Basic CSS styling was applied to change colors, fonts, and layouts. The importance of clean and semantic HTML was discussed for accessibility and SEO purposes.

As a practical task, students created a personal webpage template including their name, class, URN, and a short introduction. The session concluded with a brief overview of upcoming topics: advanced HTML tags, forms, tables, responsive design, and an introduction to JavaScript for interactivity.
